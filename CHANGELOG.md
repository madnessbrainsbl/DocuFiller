# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-08

### Added
- Initial release
- Multi-format document support (DOC, DOCX, PDF)
- Smart field detection with 14 pattern types
- LLM integration with GigaChat
- Rule-based fallback for field mapping
- OCR support for scanned PDFs
- Batch document processing
- Database management for organizations and persons
- Data card system for template data
- Document history tracking
- Cross-platform support (Windows, Ubuntu)
- Comprehensive test suite
- Example scripts
- Full documentation

### Features
- Automatic field detection in documents
- Intelligent field-to-data mapping
- Support for Russian and English languages
- Context-aware field inference
- Flexible data sources (JSON, database)
- Template reusability
- Error handling and logging

### Supported Patterns
- Double braces: `{{field}}`
- Single braces: `{field}`
- Square brackets: `[field]`
- Angle brackets: `<field>`
- Underscores: `_____`
- Dots: `...`
- Dashes: `---`
- Context markers: ФИО, Дата, Подпись, etc.
